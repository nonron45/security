# 🔐 واجهة برمجية لأمن كلمات المرور (Password Security Module API)

## 📌 المقدمة

هذا المشروع عبارة عن **واجهة برمجية (API)** تم تطويرها باستخدام إطار العمل **FastAPI** بلغة Python، ويهدف إلى تعزيز أمن كلمات المرور وتوفير أدوات حديثة لحمايتها.

يركّز النظام على تطبيق أفضل الممارسات الأمنية، مثل تقييم قوة كلمات المرور، وتوليد كلمات مرور آمنة، وإنشاء عبارات مرور (Passphrases)، بالإضافة إلى دعم التحقق الثنائي (2FA) وفحص تسريب كلمات المرور.

---

## 🎯 أهداف المشروع

يهدف هذا المشروع إلى:

* تقييم قوة كلمات المرور بناءً على معايير أمنية
* توليد كلمات مرور قوية وعشوائية
* إنشاء عبارات مرور سهلة الحفظ وآمنة
* دعم المصادقة الثنائية (2FA)
* التحقق من تسريب كلمات المرور في قواعد بيانات الاختراقات

---

## 🧱 التقنيات المستخدمة

* **Python 3**
* **FastAPI** – لإنشاء الواجهة البرمجية
* **Pydantic** – للتحقق من صحة البيانات
* **re (التعبيرات النمطية)** – لتحليل الأنماط داخل النصوص
* **secrets** – لتوليد قيم عشوائية آمنة
* **string** – للتعامل مع الحروف والأرقام
* **hashlib** – لتشفير كلمات المرور (SHA-1)
* **requests** – للتواصل مع واجهات برمجية خارجية
* **pyotp** – لإنشاء رموز التحقق الثنائية (TOTP)

---

## ⚙️ خصائص النظام

### 🔍 1. فحص قوة كلمة المرور

يقوم النظام بتحليل كلمة المرور بناءً على المعايير التالية:

* طول لا يقل عن 12 حرفًا
* احتواؤها على حروف كبيرة (A-Z)
* احتواؤها على حروف صغيرة (a-z)
* احتواؤها على أرقام (0-9)
* احتواؤها على رموز خاصة (@#$%^&*!)

📊 النتائج:

* درجة (Score)
* نسبة الأمان
* التصنيف (ضعيف / متوسط / قوي)

---

### 🔑 2. توليد كلمة مرور آمنة

* إنشاء كلمات مرور عشوائية قوية
* ضمان احتواء كلمة المرور على:

  * حروف كبيرة
  * حروف صغيرة
  * أرقام
  * رموز
* استخدام مكتبة **secrets** لضمان أمان التوليد

---

### 🧠 3. توليد Passphrase

* إنشاء كلمات مرور على شكل كلمات (مثل: apple mango lemon)
* سهلة التذكر وصعبة الاختراق
* تعتمد على أسلوب مشابه لـ Diceware

---

### 🔐 4. المصادقة الثنائية (2FA)

* إنشاء مفتاح سري (Secret Key)
* توليد رمز تحقق مؤقت (TOTP)
* متوافق مع تطبيقات مثل Google Authenticator

---

### 🛡️ 5. فحص تسريب كلمات المرور

* يعتمد على خدمة **Have I Been Pwned**
* يتحقق مما إذا كانت كلمة المرور قد تم تسريبها سابقًا
* يستخدم تقنية **k-anonymity** لحماية الخصوصية
* لا يتم إرسال كلمة المرور الكاملة

---

## 🌐 نقاط النهاية (API Endpoints)

| الطريقة | الرابط                 | الوصف               |
| ------- | ---------------------- | ------------------- |
| GET     | `/`                    | الصفحة الرئيسية     |
| POST    | `/check-password`      | فحص قوة كلمة المرور |
| GET     | `/generate-password`   | توليد كلمة مرور     |
| GET     | `/generate-passphrase` | توليد عبارة مرور    |
| GET     | `/generate-2fa`        | إنشاء 2FA           |
| POST    | `/check-breach`        | فحص التسريب         |

---

## 🧪 أمثلة الاستخدام

### ✔️ فحص كلمة المرور

```json
POST /check-password

{
  "password": "StrongPass123!"
}
```

---

### ✔️ توليد كلمة مرور

```bash
GET /generate-password?length=12
```

---

### ✔️ توليد Passphrase

```bash
GET /generate-passphrase?words=4
```

---

### ✔️ إنشاء 2FA

```bash
GET /generate-2fa
```

---

### ✔️ فحص التسريب

```json
POST /check-breach

{
  "password": "123456"
}
```
